Connect the Dots


//Create SVG element
var svg = d3.select(".scatterplot")
  .append("svg")
  .attr("width", w)
  .attr("height", h);

// Set default background fill for SVG
svg.append("rect")
  .attr("width", w)
  .attr("height", h)
  .attr("fill", "#f9f9f9");

// Add the data
svg.selectAll("circle")
  .data(dataset)
  .enter()
  .append("circle") 
  .attr("cx", function(d) { return d[1]; })  // circle x coordinate (0 index of dataset array)
  .attr("cy", function(d) { return d[2]; })  // circle y coordinate (1 index of dataset array)
  .attr("r", 20) // radius of circle
  .attr("fill", "#fff")
  • Start with the very simple scatterplot canvas
  • Fill in the background color
  • Add some data

Rapid Code Deployment

  • Capistrano (Ruby Gem) for rapid and secure deployment from specified Git branch
  • Checked in code daily which was tested by a user group the following
  • Art: guides.github.com/introduction/flow
  • Finally, one of the best ways we streamline our development is the ability to rapidly and securly push code to a web server that our clients can view and test with
  • With our Middleman Pattern Libraries we use Middleman-deploy with rsync
  • With our CMS projects we use Capistrano which is a Ruby Gem that deploys from a selected (usually master) branch of your git repository
